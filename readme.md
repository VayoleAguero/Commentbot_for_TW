Суть проекта была в написании простого бота для оставления комментариев на TradingView, причем такого, котрый способен обмануть периферийный монитор на сервере, поэтому была использована библиотека 'pyautogui' для эмуляции движений мыши.

если код не работает сразуЮ значит масштаб вашего экрана отличается от того, для которого этот код предназначался, чтобы выровнять координаты используйте код из файла 'check_coordinates'
